Upoznavanje CSS-a

Sintaksa

U narednom paragrafu pojasniti šta je šta u ovom kodu: div, p.naziv {background-color:white;}

div je selektor
p.naziv je selektor klase naziv unutar taga paragraf
background-color je atribut
white je vrijednost atributa

Selektori

CSS selektor omogućava odabir i manipulaciju HTML elementima.

ID selektor koristi ID atribut HTML taga da bi pronašao određeni element.

Selektor klase primjenjuje se na sve elemente koji imaju istu klasu unutar HTML-a.

Pozivanje selektora kroz CSS

ID selektor se poziva sa # na primjer: #naziv

Selektor klase se poziva sa . na primjer: .klasa
